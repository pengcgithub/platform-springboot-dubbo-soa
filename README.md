# platform-spring-boot-dubbo-soa

搭建一整套微服务体系的架构，主要以阿里系产品为中心，补全dubbo微服务架构。

## 基础环境

- spring-boot 2.1.1.RELEASE
- dubbo 2.6.5
- dubbo-spring-boot-starter 0.2.0
- spring-data-jpa
